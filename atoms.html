Buttons

Buttons are classes are compounding so basically all buttons are exactly the same and you can swap out colors and sizes by adding or subtracting classes. This means there can be a large variety of buttons without a huge amount of repeated code. They are also 100% CSS so there is no requests for images needed.

These classes can be applied to any element to act as a "button" for example:

<a href="#" class="btn">

Button class="" Description
btn Default standard gray button
btn btn-blue Standard blue button
btn btn-green Indicates a successful or positive action
btn btn-orange Indicates caution should be taken with this action
btn btn-red Indicates a dangerous or potentially negative action
btn btn-inverse Alternate dark gray button, not tied to a semantic action or use
btn btn-link Deemphasize a button by making it look like a link while maintaining button behavior

Icons

Scalability

These icons are font based so they can be scaled to any size.

Takes advantage of CSS styles

Any CSS styles can be applied to the icons to change their apperances.

Faster Loading

If you were to load all these icons as images on a page you'd have over 70 individual requests for files from the server. With the web font you have one request from the server that is significantly smaller than all the individual images combined.

Helpers

Helpers are classes that have no visual appearence but help to layout the page(e.g. Floating an image to the left or right).

Class Effect
.pull-left Adds float:left
.pull-right Adds float:right
.text-right Aligns child element copy right
.center Aligns child element copy center
.flush-margin Removes all margins from element
.flush-padding Removes all padding from element
.hide Hides the element
.show Shows the element by setting display:block

Responsive Utilities

These classes help us hide or show things to only certain screen widths. For example if you had a large detailed image you want to show on the desktop but doesn't really translate well to phones you may want to hide it or swap it for a different image. Be careful though because devices will still load all assets regardless of if they are hidden or not.

Class Phones 767px and below Tablets 979px to 768px Desktops Default
