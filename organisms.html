Affix

The side navigation bar is an example of the Affix plugin. It allows objects to track with the scroll bar. This is useful for keeping navigation accessible no matter where the user has scrolled.

These tags need to be added to an element to affix the element:

  <div data-spy="affix" data-offset-top="200">...</div>

There are a couple of other settings such as affix-top and affix-bottom that can be used as well.

The offset number is the number of pixels that the user must scroll to have the affix kick in. Some manipulation of the CSS might be needed to make the affixed element function correctly.
