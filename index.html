Bluebeam Framework

This is a collection of elements for piecing together pages in a simple way that adhears to the Bluebeam brand.

Atoms

The most basic building blocks of the Bluebeam website. These cannot be abstracted any further.

  • Colors
  • Typography
    • Fonts
    • Headings
    • Paragraphs
    • Blockquotes
    • Pullquotes
    • Inline Elements
    • Lists
  • Hr
  • Icons
  • Forms
    • Inputs
    • Inline Forms
  • Tables
  • Buttons
  • Helpers

Molecules

Elements that are a little more complexed than Atoms. These usually combine 2-3 Atom elements but are still fairly abstract.

  • Grid
  • Media Elements
    • Media Object
    • Media Block
  • Hero Element
  • Gallery

Organisms

Organisms are complexed elements, often need Javascript to function and have complexed markup.

  • Modal
  • Navigation
    • Navbar
    • Subnavbar
    • Nav Abstracted
    • Nav Pills
    • Nav Tabs
    • Pagination
    • Previous/Next
  • Tabs
  • Tooltips
  • Accordion
  • Carousel
  • Header
  • Footer
  • Affix

Sources & Documentation

Every single element and functionality is not contained in this documentation. I included the ones I felt would be used the most commonly used. This framework in heavily based off of Bootstrap Twitter's open source framework. They have extensive documentation and a GitHub

Here are a few other resources I used to create this:

OOCSS
Object Oriented CSS is a philosophy of writing CSS that simplifies our lives.
SMACSS - Scalable and Modular Architecture for CSS
Another way of thinking about CSS that was used to create this framework.
Boxer
A better lightbox, with a lot of flexibility and less code.
Atomic Design
A built up approach to creating websites starting with the smallest elements and working up to the largest.
